Основные теги в HTML

Теги оглавления:

TITLE

Помещает название документа в оглавление программы просмотра страниц
<title>Основные теги в HTML</title>

Теги форматирования текста:

h1

Заголовок 1-го уровня

Создает самый большой заголовок (как отдельный абзац)
<h1>Заголовок 1-го уровня</h1>

h6

Заголовок 6-го уровня

Создает самый маленький заголовок (как отдельный абзац)
<h6>Заголовок 6-го уровня</h6>

b

Жирный

Создает жирный текст
<b>Жирный</b>

i

Наклонный

Создает наклонный текст
<i>Наклонный</i>

u

Подчеркнутый

Добавляет подчеркивание к тексту.
<u>Подчеркнутый</i>

big

Шрифт

Тег <big> увеличивает размер шрифта на единицу по сравнению с обычным текстом. В HTML размер шрифта измеряется в условных единицах от 1 до 7, средний размер текста, используемый по умолчанию, принят 3. Таким образом, добавление тега <big> увеличивает текст на одну условную единицу. Допускается применение вложенных тегов <big>, при этом размер шрифта будет больше с каждым уровнем.

<big>Шрифт</big>

small

Шрифт

Тег уменьшает размер шрифта на единицу по сравнению с обычным текстом.
<small>Шрифт</small>

strong

Текст отображается жирным начертанием

Тег предназначен для акцентирования текста. Браузеры отображают такой текст жирным начертанием.
<strong>Текст отображается жирным начертанием</strong>

sub

H2O

Отображает шрифт в виде нижнего индекса. Текст при этом располагается ниже базовой линии остальных символов строки и уменьшенного размера.

H<sub>2</sub>O

sup

X2+Y=Z

Отображает шрифт в виде нижнего индекса. Текст при этом располагается ниже базовой линии остальных символов строки и уменьшенного размера.
X<sup>2</sup>+Y=Z

center

Шрифт

Тег <center> выравнивает содержимое контейнера по центру относительно родительского элемента.
<center>Шрифт</center>

font

Шрифт

Тег представляет собой контейнер для изменения характеристик шрифта, таких как размер, цвет и гарнитура. В данном присере использовали атрибут color для задания цвета шрифта.

<font color="gren">Шрифт</font>

Гиперссылки:

a

Гиперссылка

Тег редназначен для создания ссылок. В зависимости от присутствия атрибутов name или href он устанавливает ссылку или якорь. Якорем называется закладка внутри страницы, которую можно указать в качестве цели ссылки. При использовании ссылки, которая указывает на якорь, происходит переход к закладке внутри веб-страницы. Для создания ссылки необходимо сообщить браузеру, что является ссылкой, а также указать адрес документа, на который следует сделать ссылку. В качестве значения атрибута href используется адрес документа (URL, Universal Resource Locator, универсальный указатель ресурсов), на который происходит переход. Адрес ссылки может быть абсолютным и относительным. Абсолютные адреса работают везде и всюду независимо от имени сайта или веб-страницы, где прописана ссылка. Относительные ссылки, как следует из их названия, построены относительно текущего документа или корня сайта.

<a href="index.html">Шрифт</a>

Форматирование:

br

Строка1


Строка2

Вставляет перевод строки там, где находится. Не добавляет пустой отступ перед строкой.
<p>Строка1</p>
<br>
<p>Строка2</p>

ol

    Нумерованный список:
  1. Строка1
  2. Строка2

Создает нумерованный список.Каждый элемент списка должен начинаться с тега <li>.
<ol>
<li>Строка1</li>
<li>Строка2</li>
</ol>

li

Смотрите примеры для нумерованного и маркированного списков

Определяет каждый элемент списка и присваивает номер(для нумерованного списка). Предваряет каждый элемент списка и добавляет кружок или квадратик(для маркированного списка).

ul

    Маркированный список:
  • Строка1
  • Строка2

Создает маркированный список.Каждый элемент списка должен начинаться с тега <li>.
<ul>
<li>Строка1</li>
<li>Строка2</li>
</ul>

Графические элементы:

img

Тег <img> предназначен для отображения на веб-странице изображений в графическом формате GIF, JPEG или PNG. Адрес файла с картинкой задаётся через атрибут src. Если необходимо, то рисунок можно сделать ссылкой на другой файл, поместив тег <img> в контейнер <a>. При этом вокруг изображения отображается рамка, которую можно убрать, добавив атрибут border="0" в тег <img>. Рисунки также могут применяться в качестве карт-изображений, когда картинка содержит активные области, выступающие в качестве ссылок. Такая карта по внешнему виду ничем не отличается от обычного изображения, но при этом оно может быть разбито на невидимые зоны разной формы, где каждая из областей служит ссылкой.
<img src="h.jpg">

Таблицы:

table

Таблица
Название1 Название2
Элемент1 Элемент2

Элемент <table> служит контейнером для элементов, определяющих содержимое таблицы. Любая таблица состоит из строк и ячеек, которые задаются с помощью тегов <tr> и <td>.Толщина рамки таблицы задается атрибутом border.
<table border="1">Таблица
<tr>
<th>Название1</th>
<th>Название2</th>
</tr>
<tr>
<td>Элемент1</td>
<td>Элемент2</td>
</tr>
</table>

td

Таблица
Название1 Название2
Элемент1 Элемент2

Предназначен для создания одной ячейки таблицы. Тег <td> должен размещаться внутри контейнера <tr>, который в свою очередь располагается внутри тега <table>.
<table border="1">Таблица
<tr>
<th>Название1</th>
<th>Название2</th>
</tr>
<tr>
<td>Элемент1</td>
<td>Элемент2</td>
</tr>
</table>

tr

Таблица
Название1 Название2
Элемент1 Элемент2

Тег <tr> служит контейнером для создания строки таблицы. Каждая ячейка в пределах такой строки может задаваться с помощью тегаТег <tr> служит контейнером для создания строки таблицы. Каждая ячейка в пределах такой строки может задаваться с помощью тега <th> или <td>.
<table border="1">Таблица
<tr>
<th>Название1</th>
<th>Название2</th>
</tr>
<tr>
<td>Элемент1</td>
<td>Элемент2</td>
</tr>
</table>

Универсальные:

iframe

Тег <iframe> создает плавающий фрейм, который находится внутри обычного документа, он позволяет загружать в область заданных размеров любые другие независимые документы. Тег <iframe> является контейнером, содержание которого игнорируется браузерами, не поддерживающими данный тег. Для таких браузеров можно указать альтернативный текст, который увидят пользователи. Он должен располагаться между элементами <iframe> и </iframe>.
<iframe src="index.html" width="100%" height="400" align="left">
Ваш браузер не поддерживает плавающие фреймы!
</iframe>
